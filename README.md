# Kniffelblock PWA

## Dateien
- `index.html` – die App
- `manifest.json` – PWA-Konfiguration
- `sw.js` – Service Worker (Offline-Support)
- `icon-192.png` / `icon-512.png` – App-Icons

## Auf GitHub Pages hochladen

1. GitHub.com → Neues Repository `kniffelblock` (Public)
2. Alle 5 Dateien hochladen (Add file → Upload files)
3. Settings → Pages → Branch: main → Save
4. Nach 1-2 Minuten: https://DEINNAME.github.io/kniffelblock

## Auf iPhone als App installieren

1. URL in Safari öffnen
2. Teilen-Symbol → "Zum Home-Bildschirm"
3. Fertig – funktioniert wie eine echte App, auch offline!
